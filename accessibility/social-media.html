Overview

Social media is a powerful tool for communication and engagement. It's essential to follow best practices when creating social media posts to ensure that your content is accessible to all users, including those with disabilities.

Alternative text for graphics

Alternative text, or alt text, is a text description of an image that is provided when the image is not available. Alt text is not only for users who rely on screen readers but also for those with slow internet connections or who have images disabled in their browsers. Always include alternative text (alt text) for images, infographics, and other visual content.

See alternative text for more information on writing proper descriptions for your images.

Alternative text in Instagram

Computers
For new posts
  1. Upload a photo to Instagram.
  2. After choosing a filter and editing the image, choose Next.
  3. Select Accessibility, then write the alt text in the box.
  4. Select Share to post the image.
For existing posts
  1. Above your photo or video, select more_horiz Options.
  2. Select Edit.
  3. Choose Accessibility, then write the alt text in the box.
  4. Select Done to save the changes.
iPhone app
For new posts
  1. Upload a photo to Instagram.
  2. Choose a filter and edit the image. Tap Next.
  3. Tap More options.
  4. Tap Write alt text.
  5. Enter the alt text and tap Done.
For existing posts
  1. Go to the photo and tap more_horiz Options.
  2. Tap Edit.
  3. Tap Edit Alt Text in the bottom right.
  4. Enter the alt text in the box and tap Done.
Android app
For new posts
  1. Upload a photo to Instagram.
  2. Choose a filter and edit the image. Tap Next.
  3. Tap More options.
  4. Tap Write alt text.
  5. Enter the alt text and tap Done.
For existing posts
  1. Go to the photo and tap more_vert Options.
  2. Tap Edit.
  3. Tap Edit alt text in the bottom right.
  4. Enter the alt text in the box and tap check Done.

Alternative text in X

In X, alternative text is called image descriptions.

Computers
  1. After uploading an image, select Add description (underneath the image).
  2. Enter a description of the image in the text box.
    X allows a maximum description length of 1,000 characters.
  3. Select Save.
    An "ALT" badge will appear in the corner of the image.
  4. Select post.

If you add more than one image to your post, you'll see arrows at the top of the dialog that allow you to navigate to each image to add the descriptions.

iPhone app
  1. After uploading an image, select the +ALT button on the image.
  2. Enter a description of the image in the text box.
    X allows a maximum description length of 1,000 characters.
  3. Select Done.
    An "ALT" badge will appear in the corner of the image.
  4. Select post.

If you add more than one image to your post, you'll see arrows at the top of the dialog that allow you to navigate to each image to add the descriptions.

Android app
  1. After uploading an image, select the +ALT button on the image.
  2. Enter a description of the image in the text box.
    X allows a maximum description length of 1,000 characters.
  3. Select Done.
    An "ALT" badge will appear in the corner of the image.
  4. Select post.

If you add more than one image to your post, you'll see arrows at the top of the dialog that allow you to navigate to each image to add the descriptions.


Alternative text in Tik Tok

For new posts
  1. Upload or create a photo and tap Next.
  2. On the post screen, tap More options.
  3. Tap Add alternative text.
  4. Enter the alt text next to the photo(s).
  5. Tap Done.
For existing posts
  1. Tap the More options … button on the side of the photo you want to add alt text to.
  2. Tap Edit post at the bottom.
  3. Tap Add alternative text.
  4. Enter the alt text next to the photo(s).
  5. Tap Done.

Captions for videos

Captions ensure equitable access to your video's message for users with auditory disabilities, making the content understandable in noisy environments, or when sound must be muted. They are a fundamental component of digital compliance and inclusive communication.

Captions for videos in Facebook

The process for adding videos captions on Facebook varies slightly depending on whether it's a personal profile or a page (such as a business page). Pages also give you the option of auto-generating captions — these should be checked for accuracy.

SubRip (.srt) files in Facebook

Uploading a SubRip (.srt) file allows full control over the captions' timing and accuracy. SubRip (.srt) files used in Facebook must be named using the following naming convention:

[filename].[language code]_[country code].srt

For english (US) subtitles, the file name would be:

[filename].en_US.srt

For general spanish subtitles, the file name would be:

[filename].es_LA.srt

The SubRip (.srt) file format consists of a sequence of blocks. Each block is separated by a single line and the last block shouldn't have any spaces or lines after it.

Each block includes the following elements:

  1. A unique sequence number. Start with 1 and increment by 1 for each block.
  2. A time-span in the hh:mm:ss,fff --> hh:mm:ss,fff format (hours:minutes:seconds,milliseconds), indicating the start and end times for the caption.
  3. The text of the caption itself.

The start and end time markers should always use 2 digits for hours, minutes, and seconds, and 3 digits for milliseconds (for example 01:02:03,004 and not 1:2:3,004).

Example SubRip (.srt) file:

1
00:00:01,000 --> 00:00:04,000
This is the first caption.

2
00:00:05,000 --> 00:00:08,000
This is the second caption.

3
00:00:09,000 --> 00:00:12,000
This is the third caption.
Add captions to your page's video
  1. Select What's on your mind, [username]?, then Photo/Video.
  2. Choose Add photos/videos and select a video from your computer.
  3. After the video has uploaded, select Video Options on the video thumbnail.
  4. Choose Add Captions on the left, then select:
    • auto-generate captions, or
    • upload a SubRip (.srt) file
  5. Select Save, then select Post.
