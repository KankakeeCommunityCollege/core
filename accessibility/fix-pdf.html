Overview

This guide explains how to fix accessibility issues in Adobe Acrobat. Ideally, the PDF was generated from an accessible source document (see Creating accessible documents). If you don't have access to the source document you can resolve many issues in Acrobat.

Accessibility checker

Acrobat has a built in Accessibility Checker which can help you find and fix issues. Some items (like color contrast) must be checked manually.

To run the Accessibility Checker follow these steps:

  1. Open the Accessibility tools found on the right-hand side of Acrobat and select Accessibility Check.
  2. This will open an Accessibility Checker Options window.
    Use the Start Checking button with the default options.
  3. Running the checker will open the results on the left-hand side of Acrobat.
    Expand the results using the carrot symbols to see which items have passed and which have failed.

Some issues can be resolved directly in the Accessibility Checker results.
