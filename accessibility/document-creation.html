Overview

This guide explains how to make accessible Microsoft Word Documents and Adobe PDF files. Word documents are often used to generate PDF files. For these PDF documents to be accessible, they need to be created from accessible Word documents.



Accessible Templates

Microsoft Word offers accessible templates that have the following improvements:

  • Better color contrast
  • Meaningful alt text
  • Simple table structures
  • Larger font size
  • Input messages
  • And more

You can find accessible templates online or in Microsoft Word.

Online

  1. Go to create.microsoft.com (opens in new tab/window).
  2. Searching for "accessible templates".

Within Word

  1. Go to File > New.
  2. Type "accessible templates" in the Search for online templates box.

More information can be found on Microsoft's Get accessible templates for Office (opens in new tab/window) page.



Styles

Microsoft Word offers built-in styles to add headings, paragraphs, quotes, and other elements. These styles are semantic and not just visual. They tell screen readers that the text or element has a specific function.

Do not create everything as normal text and then increase font-size, boldness and/or change the font-family to make it look like a heading. Instead, you should use Word's predefined heading and other styles. If these styles don't have the look you want, you should create your own styles and/or modify existing ones. You cannot create new heading styles, but you can modify the existing ones.

To apply heading (or other) styles:

  1. Select the text.
  2. From the Home tab, in the Styles panel, select the heading (or other style) you want to apply.

Modify Heading Styles

  1. Right-click on the style you want to change and select Modify....
  2. The Modify Style window will appear where you can change the styling (Formatting).
  3. Selecting the Add to template checkbox will save your modifications to the default template used to create new documents (Normal.dotm).

Note: Modifying a style will change it throughout the whole document. For example, modifying the Heading 2 style will affect all elements that already have Heading 2 applied.



Headings

Use appropriate heading levels. This allows screen readers to create an ordered list of all headings in the document making it easier to navigate the document.

You should have only one heading 1. Ideally, your heading 1 should be the main heading for the entire document. Do not skip heading levels.
