Computers
For new posts
  1. Hover over the photo and select edit Edit.
  2. Select Alternative text in the menu on the left.
  3. Automatically generated text is shown on the left of the photo. Click Custom alt text to edit it.
  4. Write your alt text in the box.
  5. Select Save in the bottom left.
For existing posts
  1. Select the photo to open it.
  2. Select more_horiz Options in the top right and then Change Alt Text.
  3. Select Type alternative text or change the alt text in the text box. You can also select Clear to change your edited alt text back to the automatically generated text.
  4. Select Save.
