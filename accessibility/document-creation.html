Overview

This guide explains how to make accessible Microsoft Word Documents and Adobe PDF files. Word documents are often used to generate PDF files. For these PDF documents to be accessible, they need to be created from accessible Word documents.

Accessible Templates

Microsoft Word offers accessible templates that have the following improvements:

  • Better color contrast
  • Meaningful alt text
  • Simple table structures
  • Larger font size
  • Input messages
  • And more

You can find accessible templates online or in Microsoft Word.

Online

  1. Go to create.microsoft.com (opens in new tab/window).
  2. Searching for "accessible templates".

Within Word

  1. Go to File > New.
  2. Type "accessible templates" in the Search for online templates box.

More information can be found on Microsoft's Get accessible templates for Office (opens in new tab/window) page.



Styles

Microsoft Word offers built-in styles to add headings, paragraphs, quotes, and other elements. These styles are semantic and not just visual. They tell screen readers that the text or element has a specific function.

Do not create everything as normal text and then increase font-size, boldness and/or change the font-family to make it look like a heading. Instead, you should use Word's predefined heading and other styles. If these styles don't have the look you want, you should create your own styles and/or modify existing ones. You cannot create new heading styles, but you can modify the existing ones.

To apply heading (or other) styles:

  1. Select the text.
  2. From the Home tab, in the Styles panel, select the heading (or other style) you want to apply.

Modify Heading Styles

  1. Right-click on the style you want to change and select Modify....
  2. The Modify Style window will appear where you can change the styling (Formatting).
  3. Selecting the Add to template checkbox will save your modifications to the default template used to create new documents (Normal.dotm).

Note: Modifying a style will change it throughout the whole document. For example, modifying the Heading 2 style will affect all elements that already have Heading 2 applied.



Headings

Use appropriate heading levels. This allows screen readers to create an ordered list of all headings in the document making it easier to navigate the document.

You should have only one heading 1. Ideally, your heading 1 should be the main heading for the entire document. Do not skip heading levels.

Hyperlinks

See creating accessible hyperlinks for more information on creating appropriate link text.

To add hyperlinks in a word document:

  1. Select the text you want to become a link
  2. Right-click the selected link text and choose Hyperlink....
  3. This will open the Insert Hyperlink window. Add the web address you want to link to in the Address field.

    You can also adjust the link text using the Text to Display field.


Lists

Lists are great for grouping and organizing information. Use ordered lists when the order of the list items is important (e.g. step-by-step instructions). Use unordered lists when the order does not matter (e.g. a list of ingredients). See using lists within accessible content for more information on using lists.

To add lists in a document:

  1. On the Home tab, in the Paragraph panel, choose Bulleted () for unordered lists, Numbered () for ordered lists, and Multilevel Lists () for multilevel ordered and unordered lists.
  2. Use the little carrot next to the list icon to select different list styles for each type of list.


Alternative Text

Images

If you insert an image into your document, it must have alternative text (alt-text). The only exception is when the graphic is purely decorative and does not add meaningful info needed to understand the content. Without alt-text, screen reader users will not be able understand what your image or graphic is about.

For more information and best practices on creating helpful and appropriate alternative text, see how to use alternative text to aid understanding.

To add alternative text to images:

  1. Right-click the image or graphic and select View Alt Text...
    • On newer versions of Microsoft Word, it may attempt to automatically generate alternative text when you insert an image. This AI generated alt-text usually needs to be adjusted. Clicking on the auto-generated alt-text will also open the Alt Text window.
  2. This opens the Alt Text panel to the right side of the document.
    Add (or edit) the alternative text using the text box.
    • You can also activate the Mark as decorative toggle switch for images that are purely decorative and don't add meaning to the content.

Tables

Do not use a screenshot or picture of a table; you should create it as content in the page. You can add alternative text to tables, which can help screen reader users in understanding the table and its context.

To add alt-text to a table:

  1. Right-click anywhere on the table and choose Table Properties....
  2. This opens the Table Properties window. Choose the Alt Text tab.
  3. Enter a Title and Description for the table and then select the OK button.

Charts and Graphs

The information contained in charts and graphs also need to be conveyed in a way that non-visual users can understand. Whenever possible, you should include a simple table underneath it containing the data.

If you cannot add the data in a table, you need to add alternative text. Describe the chart or graph like you're explaining it to a friend who cannot see it. Include any data contained within the chart or graph and any other important details.

Other graphics

Other forms of graphics like shapes, icons, 3D models, SmartArt, and charts require alternative text too.

Add alt-text to these types of graphics just like you would for images. Right-click the graphic and select Edit Alt Text...

Grouped graphics

If you use multiple icons, shapes, or other objects to create a single graphic (e.g. flow-chart or node map), it still needs alternative text. Since the various objects are used to create a single graphic, they need to be grouped so that you can add alt text to your design as a whole.

To group multiple objects:

  1. Select all the objects. You can select the first object and then hold the shift key add more to your selection.
  2. With all the objects selected, navigate to the Shape Format (or Picture Format tab depending on what kind of objects you selected).
    Open the Group Objects dropdown () and select Group.
  3. Now that all the shapes are grouped, you can add alternative text like you would with an image.
    Right-click the grouped objects and select View Alt Text.


Tables and table headers

It is especially important that tables in documents not be overly complex. Ideally, you should have a single header row and no merged cells anywhere. Empty cells should also be avoided â€” use values like n/a instead. If a table is too complex, you can usually break it up into multiple simple tables.

Header row

To make sure your table has a header row:

  1. Click anywhere in the table and activate the Table Design tab.
  2. Make sure the Header Row checkbox is active.

Header column

A simple table can also contain a header column as its first column. To add a header column:

  1. Click anywhere in the table and activate teh Table Design tab.
  2. Make sure the First Column checkbox is active.


Spacing

Improperly positioned text blocks in Microsoft Word using the tab, spacebar, and enter keys can cause screen readers to announce all the extra spacing characters. These keys are often used to adjust layouts by adding horizontal and vertical spacing and indentation. This can make the document difficult to understand for those users.

Microsoft Word has multiple built-in features for adjusting spacing and indentation in your document layout. You should use these instead.

The horizontal spacing tools include the Decrease Indent () and Increase Indent () tools. The vertical spacing tools include the Line and Paragraph Spacing () dropdown which has multiple options. These can be found in the Home tab of Word, within the Paragraph panel.

Horizontal spacing (indentation)

To adjust horizontal spacing (i.e. add or remove indentation):

  1. Select the text you want to indent (or remove indentation from).
  2. Navigate to the Home tab and find the Paragraph panel.
  3. Use the Increase Indent () tool to add horizontal spacing (indentation).
    You can increase the indentation multiple times to further position your text.
  4. To decrease or remove horizontal spacing, use the Decrease Indent () tool while the text you want to adjust is highlighted.
Custom indentation

To customize the amount of indentation, you can use Line Spacing Options... found in the Line and Paragraph Spacing dropdown:

  1. Select the text you want to adjust.
  2. Navigate to the Home tab and find the Paragraph panel.
    Open the Line and Paragraph Spacing dropdown and select Line Spacing Options...:
  3. This will open the Paragraph window.
    Under Indentation use the Left and Right fields to specify indentation values in inches.

Vertical spacing

You can adjust line and paragraph spacing using the tools found in the Line and Paragraph Spacing () dropdown.

Line spacing

To adjust line spacing:

  1. Select the text you want to adjust.
  2. Navigate to the Home tab and find the Paragraph panel.
    Open the Line and Paragraph Spacing dropdown and select the line spacing value you want (e.g. 1.0, 1.15, 1.5, 2.0, etc.).

The line spacing values adjust the line-height relative to the text-hight. For example, 2.0 line spacing will make each line two-times the text height.

Paragraph spacing

To adjust paragraph spacing (i.e. the amount of space before and after paragraphs):

  1. Select the paragraph(s) you want to adjust and navigate to the Home tab and find the Paragraph panel.
  2. Open the Line and Paragraph Spacing dropdown:
    • You can use Add Space Before Paragraph and Remove Space After Paragraph, or...
    • use Line Spacing Options... for more granular control.

      This will open the Paragraph window where you can specify exact values for paragraph spacing.
      Modify the Before and After fields to adjust the spacing before and after paragraphs.
