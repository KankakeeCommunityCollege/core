Overview

This guide explains how to fix accessibility issues in Adobe Acrobat. Ideally, the PDF was generated from an accessible source document (see Creating accessible documents). If you don't have access to the source document you can resolve many issues in Acrobat.

Accessibility Checker

Acrobat has a built in Accessibility Checker which can help you find and fix issues. Some items (like color contrast) must be checked manually.

To run the Accessibility Checker follow these steps:

  1. Open the Accessibility tools found on the right-hand side of Acrobat and select Accessibility Check.
  2. This will open an Accessibility Checker Options window.
    Use the Start Checking button with the default options.
  3. Running the checker will open the results on the left-hand side of Acrobat.
    Expand the results using the carrot symbols to see which items have passed and which have failed.

Some issues can be resolved directly in the Accessibility Checker results.
For example, if you see a Tab order - Failed issue, you can simply right-click it and select Fix from the right-click menu.

You can also right-click an issue and choose Explain and you will be taken to an Acrobat support page that offers an explanation of the issue an how to fix it.

Document title

A missing document title is one of the most common issues when evaluating PDF accessibility. The document title is what appears for users in search results. The title should be unique. For example, if I have a bunch of agenda PDF files and they all have the title "Agenda" it will make finding a specific file very difficult. All of my search results will show the same title for each. Instead, a title like "Meeting Agenda for October 20, 2025" would be much more helpful.

When creating a PDF from a Word document, it's best to add the document title in the source file before generating the PDF.
Learn how to add document title to a word document.

A good indicator that you're missing a title is seeing the filename at the top of your open PDF file.

To add or adjust your document title in Adobe Acrobat, do the following:

  1. Open the File menu and select Properties....
  2. This will open the Document Properties window.
    Activate the Description tab.
  3. Enter your unique title in the Title field then use the OK button to commit your change.
  4. Don't forget to save your file (Win: Ctrl + S or Mac: âŒ˜ command + S) after modifying the document title.


Document language

PDF files also need a document language set for accessibility.

When creating a PDF from a Word document, it's best to add the document language in the source file before generating the PDF.
Learn how to add document language to a word document.

You can set or modify the document language in the Document Properties:

  1. Open the File menu and select Properties....
  2. This will open the Document Properties window.
    Activate the Advanced tab.
  3. Use the Language field to specify the document language.
    Either select the language from the dropdown menu or type into the field.
  4. Use the OK button to commit your change.
  5. Don't forget to save your file (Win: Ctrl + S or Mac: âŒ˜ command + S) after modifying the document language.

Tagging a PDF file

Tags represent the elements and structure of a document. They help screen readers interpret the document properly. Ideally, the source document was setup and exported correctly and your PDF already has tags (see Creating accessible documents). However, if you don't have access to the source document there are a couple of methods for tagging in Acrobat.

Autotag Document

Acrobat has an Autotag Document tool which does its best to identify and tag the various elements. You will need to check the tag structure and sometimes make adjustments.
